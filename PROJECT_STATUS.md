## 🎮 ゲーム概要
「脳を鍛える大人のDSトレーニング」の計算100/計算20モードを再現したブラウザゲーム。

- **計算100モード**: 全100問
- **計算20モード**: 全20問
- **入力方式**: キーボード（数字パッド）/ 手書き（Canvas）
- **不正解 → +5秒ペナルティ → 次の問題へ進む**（再挑戦なし）
- **タイマーは結果画面のみ表示**
- **10問ごとにマイルストーン表示**（ノンブロッキング）
- **スタート時カウントダウン** ③→②→① 表示
- **正解時はノータイムで次の問題へ**

## ✅ 現在の実装状態

### レイアウト（DS本家風）
- **左パネル: 固定3スロット表示**
  - 上スロット: 前の問題（解答済み、`7-5＝2 ⭕` 形式）
  - 中央スロット: 現在の問題（**黒枠で囲む**）
  - 下スロット: 次の問題（予告）
  - スクロール不要・常に3問のみ表示
  - **解答後スライドアニメーション**（下から上）
- **右パネル: 入力のみ（画面いっぱい）**
  - 問題式は右に表示しない
  - キーボードモード: 数字入力欄 + 数字パッド + 書き直しボタン（**全て横幅いっぱい**）
  - 手書きモード: Canvasのみ + 書き直しボタン（**横幅いっぱい拡大**）
  - 右下に「やめる」ボタン（フッターと被らないよう調整済み）
- **フッター**: `made by hiro / ヒロ | GitHub: h1ro223`（fixed bottom）

### カウントダウン
- スタートボタン押下 → ③→②→①（各0.8秒） → 0.5秒待機 → ゲーム開始
- 手書きモード: モデル読み込み → 準備完了 → カウントダウン
- **問題画面表示後にタイマー開始**

### キーボード入力
- 正解の数字を入力した瞬間 → **即時正解判定・次へ（ノータイム）**
- 答えと同桁数入力で不正解 → **即時不正解判定・次へ**
- 途中入力（桁数が足りない）→ **2秒後に自動判定**
- Enterキーでも即時判定

### 手書き入力
- `onnxruntime-web@1.17.0` CDNで読み込み
- **ONNXモデル** 読み込み試行 → 失敗時は**テンプレートマッチング**にフォールバック
- フォールバック: Canvas上で0-9をフォント描画してコサイン類似度で照合
- 処理: 28x28リサイズ → グレースケール反転正規化 → 重心(14,14)移動（MNIST準拠）
- **ストローク終了時に即認識** → 正解なら即判定、不正解なら2秒猶予で書き直し可能

### 問題仕様
- 足し算: 各数値0〜19
- 引き算: 結果が0以上（a≥b）
- 掛け算: 1〜9の九九
